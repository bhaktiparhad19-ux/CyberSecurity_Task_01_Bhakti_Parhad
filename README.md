# Cyber Security Task 01 – Personal Security Audit & Cyber Awareness Assessment

## 📌 Objective

The objective of this task is to understand the fundamentals of cybersecurity by evaluating personal digital security practices, identifying potential risks, and learning how to protect devices and online accounts using cybersecurity best practices.

---

## 📂 Task Overview

This repository contains the completed work for **Cyber Security Task 01**, which includes the following sections:

### Part A – Device Security Assessment

* Reviewed the security status of my personal laptop.
* Verified the operating system and OS version.
* Checked Windows Update status.
* Confirmed antivirus protection.
* Verified firewall configuration.
* Included screenshots as evidence.

### Part B – Password Security Review

* Explained the characteristics of a strong password.
* Discussed the importance of using unique passwords.
* Explained Multi-Factor Authentication (MFA).
* Identified the risks of password reuse.
* Created a password security policy based on cybersecurity best practices.

### Part C – Online Account Security Check

* Reviewed the security settings of three online accounts.
* Verified the use of Multi-Factor Authentication (MFA).
* Confirmed the use of strong passwords.
* Checked recovery email configuration.
* Summarized the findings in a comparison table.

### Part D – Cyber Threat Research

* Researched and explained the following cyber threats:

  * Phishing
  * Malware
  * Ransomware
  * Social Engineering
  * Data Breach
* Included definitions, real-world examples, and prevention methods for each threat.

### Part E – Cyber Security Awareness Poster

* Designed a cybersecurity awareness poster on **Social Engineering Awareness** using Canva.
* Highlighted common attack techniques and best practices for staying safe online.

### Part F – Security Reflection

* Reflected on my current cybersecurity practices.
* Identified potential risks in my digital habits.
* Listed security improvements to implement.
* Explained the importance of cybersecurity in today's digital world.
